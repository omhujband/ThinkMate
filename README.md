# ThinkMate

ThinkMate is an on-device AI study assistant designed to help users learn and retain information more effectively.

## Vision

The project aims to leverage local AI capabilities to provide a personalized learning companion that respects user privacy and functions offline.

## Getting Started

1.  **Install Flutter** (3.10 or higher).
2.  **Clone the repository**.
3.  **Run `flutter pub get`** to install dependencies.
4.  **Run the app** using `flutter run`.
